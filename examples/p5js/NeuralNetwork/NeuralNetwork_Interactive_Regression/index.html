Interactive Regression Example - Neural Network

Instructions

  1. Add data by clicking inside the canvas
  2. Edit learning rate = amount that the weights are updated during training
  3. Edit number of hidden units = number of hidden units in dense layer
  4. Edit number of training epochs = number of iterations over entire dataset
  5. Edit number of batch size = number of data points to work through before updating the internal model parameters
  6. Click on the Train button
